# R-036 - Lista de 30 posibles clientes

## Objetivo

Crear una lista inicial de posibles clientes o perfiles de cliente para Vigex.

Esta lista no contiene datos reales de empresas ni información personal. Se plantea como una base comercial inicial para identificar sectores donde Vigex podría aportar valor.

## Enfoque

Vigex está pensado principalmente para microempresas y PYMES que:

- Tienen datos importantes.
- Usan bases de datos o aplicaciones internas.
- No tienen un departamento técnico propio.
- Realizan backups de forma manual o poco controlada.
- Necesitan logs, alertas y soporte sencillo.
- Prefieren una solución local o híbrida antes que depender totalmente de la nube.

## Criterios para seleccionar posibles clientes

Un posible cliente encaja bien si cumple varios de estos criterios:

- Tiene entre 5 y 30 empleados.
- Usa algún software de gestión, facturación, TPV, reservas o base de datos.
- Tiene miedo a perder información.
- No revisa restauraciones periódicamente.
- No tiene un sistema claro de alertas.
- Depende de una persona concreta para revisar copias.
- Quiere soporte cercano y sencillo.
- No dispone de personal técnico interno.

## Priorización inicial

Para una primera prueba comercial, se recomienda priorizar:

1. Gestorías.
2. Asesorías.
3. Talleres.
4. Clínicas pequeñas.
5. Tiendas con TPV local.
6. Academias.
7. Empresas con ERP o base de datos local.
8. Proveedores IT pequeños que puedan actuar como partner.

## Clientes más fáciles para un primer piloto

Los perfiles más adecuados para un primer piloto técnico son:

- Empresa conocida o cercana.
- Negocio con pocos servidores.
- Cliente que ya tenga una base de datos local.
- Cliente que acepte una instalación controlada.
- Cliente que no exija disponibilidad crítica desde el primer día.
- Cliente que permita probar backup, logs y restauración en entorno seguro.

## Clientes que conviene evitar al principio

Para las primeras pruebas se recomienda evitar:

- Empresas grandes con requisitos legales complejos.
- Entornos médicos muy críticos sin supervisión profesional.
- Empresas que exijan soporte 24/7 desde el inicio.
- Clientes sin copia externa posible.
- Clientes que no permitan pruebas de restauración.
- Clientes que pidan garantías absolutas de recuperación.

## Mensaje comercial base

Vigex está pensado para empresas pequeñas que quieren saber si sus copias de seguridad funcionan, tener registros claros de actividad y recibir avisos cuando algo falla, sin tener que montar herramientas complejas ni depender de procesos manuales.

## Resultado esperado

Esta lista sirve como punto de partida para buscar pilotos, preparar llamadas comerciales y enfocar la presentación del producto hacia sectores donde la necesidad de backups, logs y soporte es clara.
