# Validación R-034 - Manual rápido para cliente

## Objetivo de la validación

Comprobar que existe un manual rápido entendible para un cliente no técnico o con conocimientos básicos.

## Comprobaciones realizadas

| Comprobación | Resultado |
|---|---|
| Explica qué es DASC Server Manager | Sí |
| Explica cómo acceder al panel | Sí |
| Explica la pantalla principal | Sí |
| Explica el módulo de copias | Sí |
| Explica los tipos de backup | Sí |
| Explica el uso de logs | Sí |
| Explica servicios | Sí |
| Explica alertas | Sí |
| Explica usuarios y permisos | Sí |
| Incluye buenas prácticas | Sí |
| Incluye qué hacer ante errores | Sí |
| Diferencia responsabilidad del cliente y del proveedor | Sí |

## Evidencias recomendadas

Para validar esta tarea en una entrega o presentación, se puede mostrar:

1. Archivo `docs/cliente/R-034_manual_rapido_cliente.md`.
2. Captura del manual abierto en GitHub.
3. Explicación breve de que está orientado a cliente no técnico.

## Resultado final

R-034 queda documentada cuando el manual rápido existe en el repositorio y puede ser consultado desde GitHub.
