# Validación R-046 - Recalcular costes reales

## Objetivo

Validar que los costes reales estimados de DASC Server Manager han sido revisados después de los pilotos técnicos y de la definición del SLA.

## Resultado general

R-046 se considera validada.

Se han recalculado los costes de instalación, operación mensual, infraestructura y soporte para los perfiles Lite, PyME estándar y Pro.

## Comprobaciones

| Comprobación | Resultado |
|---|---|
| Coste de instalación definido | OK |
| Coste mensual de operación definido | OK |
| Coste de infraestructura estimado | OK |
| Coste por perfil definido | OK |
| Margen aproximado analizado | OK |
| Relación con SLA revisada | OK |
| Relación con pilotos revisada | OK |
| Perfil recomendado identificado | OK |
| Costes excluidos documentados | OK |

## Precios orientativos validados

| Perfil | Instalación | Mensualidad recomendada |
|---|---:|---:|
| Lite | 150-250 € | 49-79 €/mes |
| PyME estándar | 300-500 € | 99-179 €/mes |
| Pro | 600-1.000 € | 249-399 €/mes |

## Perfil recomendado

El perfil recomendado para la fase inicial es:

    PyME estándar

## Justificación

El perfil PyME estándar se considera el más equilibrado porque:

- Tiene mayor separación técnica que Lite.
- Ofrece más valor al cliente.
- Permite mejor margen.
- Encaja mejor con el SLA realista.
- Reduce el riesgo frente a concentrar todo en un único servidor.
- Ya ha sido validado en el piloto 1.

## Limitaciones

Los importes son orientativos.

Deben revisarse antes de una venta real según:

- proveedor de infraestructura;
- tamaño de la base de datos;
- volumen de backups;
- horario de soporte;
- frecuencia de informes;
- criticidad del cliente;
- necesidad de copia externa real;
- necesidad de restauraciones garantizadas.

## Criterio de cierre

R-046 se cierra porque:

- Existen precios orientativos por perfil.
- Se ha estimado instalación y mensualidad.
- Se ha comparado coste y margen.
- Se ha definido el perfil recomendado.
- Los costes quedan alineados con el SLA.
- Los pilotos han servido como base para ajustar la estimación.
