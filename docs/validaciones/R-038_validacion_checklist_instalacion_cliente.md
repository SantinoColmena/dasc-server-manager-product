# Validación R-038 - Checklist de instalación en cliente

## Objetivo de la validación

Confirmar que existe un checklist de instalación que permite preparar, ejecutar y cerrar una instalación de DASC Server Manager en un cliente.

## Comprobaciones realizadas

| Comprobación | Resultado |
|---|---|
| Incluye información previa del cliente | Sí |
| Incluye información técnica previa | Sí |
| Incluye requisitos mínimos | Sí |
| Incluye comprobaciones de seguridad | Sí |
| Incluye instalación de base de datos | Sí |
| Incluye instalación de backups y servicios | Sí |
| Incluye instalación del panel DASC | Sí |
| Incluye configuración inicial | Sí |
| Incluye validación funcional | Sí |
| Incluye validación de copia externa | Sí |
| Incluye evidencias a guardar | Sí |
| Incluye entrega al cliente | Sí |
| Define criterios de cierre | Sí |

## Evidencias recomendadas

Para validar esta tarea, se puede mostrar:

1. Archivo `docs/cliente/R-038_checklist_instalacion_cliente.md`.
2. Captura del checklist abierto en GitHub.
3. Relación con los scripts de instalación ya existentes.
4. Explicación de que el checklist sirve para futuras instalaciones reales o pilotos.

## Resultado final

R-038 queda documentada cuando el checklist existe en el repositorio y puede usarse como guía de instalación en cliente.
