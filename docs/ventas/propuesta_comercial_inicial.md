# Propuesta comercial inicial - DASC Server Manager

## 1. Resumen

DASC Server Manager es una solución local para pequeñas empresas que necesitan controlar sus copias de seguridad, restauración, logs, servicios y alertas sin depender de procesos manuales difíciles de revisar.

El objetivo no es sustituir toda la administración informática de la empresa, sino aportar una capa sencilla de control y tranquilidad sobre datos importantes.

## 2. Problema detectado

Muchas pequeñas empresas tienen alguno de estos problemas:

- No saben si sus backups se están haciendo correctamente.
- No prueban restauraciones.
- Dependen de una persona concreta para tareas críticas.
- No tienen logs claros de acciones importantes.
- No reciben alertas cuando algo falla.
- No tienen un informe mensual del estado de sus copias.

## 3. Solución propuesta

Instalar DASC Server Manager como panel local para:

- Ejecutar y revisar copias de seguridad.
- Consultar historial de backups.
- Validar restauraciones de prueba.
- Registrar acciones relevantes.
- Controlar servicios básicos.
- Recibir alertas.
- Generar una revisión mensual del estado.

## 4. Modalidad recomendada

### DASC PyME Inicial

Pensado para una primera implantación controlada en una pequeña empresa.

Incluye:

- Revisión inicial del entorno.
- Instalación del panel.
- Configuración de backup.
- Configuración de logs.
- Prueba de restauración.
- Configuración de alertas básicas.
- Informe inicial.
- Soporte limitado durante el primer mes.

## 5. Precio orientativo

Instalación inicial:

- Entre 250 € y 500 €, según dificultad del entorno.

Mantenimiento mensual opcional:

- Básico: 39 €/mes.
- Estándar: 59 €/mes.
- Gestionado inicial: 79 €/mes.

El precio definitivo depende de:

- Número de servidores.
- Tipo de base de datos.
- Tamaño de los backups.
- Necesidad de copia externa.
- Horario de soporte.
- Nivel de revisión mensual.

## 6. Límites del servicio

El servicio no incluye por defecto:

- Soporte ilimitado.
- Administración completa de toda la infraestructura.
- Intervenciones fuera del horario acordado.
- Recuperaciones críticas no probadas previamente.
- Compra de hardware.
- Responsabilidad sobre sistemas externos no revisados.
- Cumplimiento legal automático.

## 7. Condiciones recomendadas para el primer cliente

Para el primer cliente o piloto de pago se puede aplicar un precio reducido si acepta:

- Dar feedback real.
- Permitir documentar el caso de uso sin datos sensibles.
- Validar una restauración de prueba.
- Usar la solución con alcance controlado.
- No tratar la release candidate como versión final cerrada.

## 8. Próximo paso

Realizar una revisión inicial del entorno y confirmar si DASC Server Manager encaja con la infraestructura del cliente.
