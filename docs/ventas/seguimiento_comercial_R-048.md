Registrar contactos reales con posibles clientes para validar la primera venta de DASC Server Manager.

## Estados posibles

- Pendiente.
- Contactado.
- Interesado.
- Reunión pendiente.
- Oferta enviada.
- Aceptado.
- Rechazado.
- No responde.

## Objeciones a registrar

Cuando un cliente no acepte, se debe registrar el motivo real:

- Precio alto.
- No entiende el producto.
- Ya tiene proveedor.
- No tiene servidor propio.
- No le preocupa el problema.
- Le interesa solo backup.
- Le interesa solo soporte.
- Quiere esperar.
- No confía en una versión candidata.
- Otro motivo.

## Criterio de avance

R-048 avanzará cuando al menos exista:

- 1 cliente interesado con oferta enviada, o
- 3 contactos reales con feedback documentado.

## Criterio de cierre

R-048 se cerrará cuando exista:

- 1 cliente pagando, o
- feedback suficiente para ajustar la oferta antes de seguir vendiendo.
