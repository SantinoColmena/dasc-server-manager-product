# Propuesta de valor comercial - Vigex

## 1. Qué es Vigex

Vigex es una herramienta local pensada para pequeñas y medianas empresas que necesitan controlar sus copias de seguridad, servicios, logs y alertas desde un panel web sencillo.

El producto permite centralizar tareas básicas de administración de servidores sin depender completamente de soluciones externas, servicios cloud complejos o procesos manuales difíciles de controlar.

La idea principal es que una empresa pueda tener más control sobre sus datos, sus copias de seguridad y sus operaciones críticas desde una interfaz clara y accesible.

---

## 2. Problema que resuelve

Muchas pequeñas empresas tienen bases de datos, servidores o servicios internos, pero no siempre cuentan con un sistema claro para revisar si sus copias de seguridad se están haciendo correctamente.

En muchos casos, los backups se realizan de forma manual, no se comprueban con frecuencia o dependen de una única persona técnica. Esto puede generar riesgos importantes para la empresa.

Los problemas principales que resuelve Vigex son:

- Falta de control sobre las copias de seguridad.
- Riesgo de pérdida de datos.
- Dependencia de tareas manuales.
- Dificultad para comprobar si un backup se ha realizado correctamente.
- Falta de historial claro de acciones.
- Poca visibilidad sobre quién ha realizado cada operación.
- Dificultad para gestionar servicios básicos del servidor.
- Dependencia de soluciones demasiado complejas para una PyME.

---

## 3. Cliente objetivo

Vigex está pensado principalmente para pequeñas y medianas empresas que trabajan con servidores propios, bases de datos locales o servicios internos.

El cliente ideal no necesariamente dispone de un gran departamento técnico, pero sí necesita proteger sus datos y tener una forma sencilla de revisar el estado de sus sistemas.

Ejemplos de clientes objetivo:

- Gestorías.
- Academias.
- Pequeñas tiendas.
- Despachos profesionales.
- Empresas con base de datos MySQL o MariaDB.
- Negocios con servidor local.
- Empresas que quieren reducir dependencia de servicios externos.
- PyMEs que necesitan una solución sencilla de backups y trazabilidad.

---

## 4. Propuesta de valor

Vigex permite a una empresa controlar sus backups, logs, servicios y alertas desde un único panel local, reduciendo errores humanos y mejorando la visibilidad sobre las operaciones del servidor.

La propuesta de valor se basa en cuatro ideas principales:

### Control local

El sistema puede instalarse en la infraestructura del cliente, evitando depender totalmente de plataformas externas.

### Simplicidad

El panel está pensado para que las acciones principales se puedan realizar desde una interfaz sencilla, sin obligar al usuario a ejecutar comandos manualmente.

### Seguridad operativa

El sistema ayuda a reducir riesgos mediante backups, logs de actividad y control básico de acciones realizadas en el panel.

### Adaptabilidad

La solución puede adaptarse a diferentes escenarios según los recursos del cliente:

- 1 servidor para instalaciones simples.
- 2 servidores para una separación básica entre sistema principal y backups.
- 3 servidores para una arquitectura más completa y profesional.

---

## 5. Beneficios para el cliente

Los beneficios principales para el cliente son:

- Mayor control sobre sus copias de seguridad.
- Menos dependencia de procesos manuales.
- Reducción del riesgo de pérdida de datos.
- Panel centralizado para tareas básicas.
- Historial de acciones realizadas.
- Posibilidad de controlar servicios desde la interfaz.
- Mejor trazabilidad de operaciones.
- Solución más accesible para pequeñas empresas.
- Instalación adaptable a los recursos disponibles.

---

## 6. Diferencia frente a otras soluciones

Existen soluciones de backup, monitorización y administración más avanzadas, pero muchas veces están pensadas para empresas grandes, requieren conocimientos técnicos elevados o tienen costes recurrentes altos.

Vigex se diferencia porque busca ser:

- Más sencillo de entender.
- Más directo para una PyME.
- Instalado en local.
- Adaptable a infraestructuras pequeñas.
- Centrado en backups, logs y servicios básicos.
- Más económico que soluciones empresariales complejas.
- Menos dependiente de plataformas externas.

No pretende sustituir herramientas empresariales avanzadas de ciberseguridad, sino ofrecer una solución práctica para empresas que necesitan empezar a controlar mejor sus datos y operaciones.
