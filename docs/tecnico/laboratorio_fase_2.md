\# Decisiones de producto



\## Decisión - Laboratorio de Fase 2



Para la Fase 2 Seguridad y Restauración se trabajará con un laboratorio doble.



Docker se utilizará para desarrollo rápido y pruebas repetibles. Las tres máquinas Ubuntu se utilizarán como validación final real, especialmente para instaladores, SSH, systemd, permisos, MariaDB, backups y restauración.



La arquitectura de 2 servidores se mantiene como objetivo estándar PyME posterior, pero no bloqueará el avance inicial de la Fase 2. La arquitectura de 1 servidor queda como perfil Lite futuro y solo será aceptable con copia externa obligatoria.
