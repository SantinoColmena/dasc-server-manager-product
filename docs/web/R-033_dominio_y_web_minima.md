# R-033 - Dominio y web mínima

## Objetivo

Preparar la base de una web mínima para presentar DASC Server Manager como producto-servicio orientado a PYMES.

Esta tarea no implica comprar obligatoriamente un dominio real durante la fase académica. El objetivo es dejar definido el enfoque comercial, la estructura de la web y los contenidos mínimos para una futura publicación.

## Estado de dominio

Compra de dominio real: Pendiente

Motivo:

Durante la fase de presentación final no es necesario realizar un gasto real ni publicar una web definitiva. Se deja preparada la propuesta de dominio y el contenido mínimo para poder activarlo más adelante si el proyecto continúa.

## Posibles dominios

Opciones recomendadas:

| Dominio propuesto | Comentario |
|---|---|
| dascservermanager.com | Nombre completo del producto |
| dascservermanager.es | Enfoque mercado español |
| dascbackup.es | Más comercial y fácil de recordar |

## Dominio recomendado

Para una primera versión comercial, se recomienda:

`dascpyme.es`

Motivos:

- Es corto.
- Es fácil de recordar.
- Encaja con el cliente objetivo.
- Comunica que está pensado para pequeñas y medianas empresas.
- No limita el producto solo a backups.

Alternativa más técnica:

`dascservermanager.es`

## Objetivo de la web mínima

La web mínima debe servir para:

- Explicar qué es DASC.
- Mostrar el problema que resuelve.
- Presentar los módulos principales.
- Explicar los paquetes Lite, PyME y Pro.
- Ofrecer contacto para demo o piloto.
- Dar confianza al cliente.

## Texto principal de la web

### Título

DASC Server Manager

### Subtítulo

Backups, logs, alertas y soporte sencillo para PYMES.

### Mensaje inicial

DASC Server Manager ayuda a pequeñas y medianas empresas a controlar sus copias de seguridad, revisar logs, recibir alertas y mantener una visión clara del estado de sus servicios desde un panel sencillo.

No es solo un panel técnico: es una solución acompañada de instalación, revisión y soporte.

## Sección: Problema

Muchas empresas pequeñas realizan copias de seguridad, pero no siempre saben si esas copias funcionan, si se pueden restaurar o si alguien está revisando los errores.

DASC nace para reducir esa incertidumbre.

Problemas habituales:

- Backups manuales.
- Copias sin validar.
- Falta de alertas.
- Logs poco claros.
- Dependencia de una sola persona.
- Riesgo de pérdida de datos.
- Dificultad para restaurar en caso de incidente.

## Sección: Solución

DASC centraliza en un único panel las operaciones esenciales para proteger los datos de una PyME.

Permite:

- Ejecutar copias de seguridad.
- Revisar historial de backups.
- Consultar logs.
- Gestionar servicios.
- Recibir alertas.
- Controlar usuarios y permisos.
- Preparar restauraciones de forma más segura.

## Sección: Funcionalidades

### Copias de seguridad

Backups completos, incrementales y diferenciales según el entorno configurado.

### Logs

Registro de accesos, acciones y resultados para saber qué ha ocurrido en el sistema.

### Alertas

Avisos ante errores de backup, fallos de servicios o incidencias importantes.

### Servicios

Vista básica del estado de servicios y acciones controladas para usuarios autorizados.

### Usuarios y permisos

Cada usuario accede solo a las secciones que necesita.

### Soporte

Instalación, revisión y mantenimiento como parte del servicio.

## Sección: Demo

Texto recomendado:

Solicita una demo para ver cómo DASC puede ayudarte a controlar backups, logs y alertas desde un panel sencillo.

La demo se realiza con datos de prueba y sin acceso a información real del cliente.

## Sección: Preguntas frecuentes

### ¿DASC funciona en local?

Sí. Está pensado para funcionar en infraestructura local o híbrida.

### ¿Mis datos tienen que ir a la nube?

No obligatoriamente. Se puede trabajar en local y añadir copia externa si el cliente lo necesita.

### ¿DASC sustituye a un técnico?

No completamente. DASC facilita la gestión, pero las acciones críticas deben supervisarse.

### ¿Puedo probarlo antes?

Sí. La idea es ofrecer una demo o piloto técnico controlado.

### ¿Está pensado para grandes empresas?

No es el objetivo inicial. DASC está pensado principalmente para microempresas y PYMES.
